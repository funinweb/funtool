# `snakeCase`

Converts a string into `snake_case` format.

## Example

```ts
import { snakeCase } from 'funtool/string'

snakeCase('hello world')
// => 'hello_world'
```

## Parameters

- `input` (`string`): The string to transform.

## Returns

- `string`: The snake_cased string.
