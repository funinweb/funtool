# `kebabCase`

Converts a string into `kebab-case` format.

## Example

```ts
import { kebabCase } from 'funtool/string'

kebabCase('hello world')
// => 'hello-world'
```

## Parameters

- `input` (`string`): The string to transform.

## Returns

- `string`: The kebab-cased string.
