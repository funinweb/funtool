# `upperCase`

Converts all characters in a string to uppercase.

## Example

```ts
import { upperCase } from 'funtool/string'

upperCase('hello')
// => 'HELLO'
```

## Parameters

- `input` (`string`): The string to convert.

## Returns

- `string`: The uppercase string.
