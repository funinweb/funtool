# `capitalize`

Capitalizes the first character of the input string.

## Example

```ts
import { capitalize } from 'funtool/string'

capitalize('hello')
// => 'Hello'
```

## Parameters

- `input` (`string`): The string to capitalize.

## Returns

- `string`: The capitalized string.
