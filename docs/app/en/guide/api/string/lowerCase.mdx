# `lowerCase`

Converts all characters in a string to lowercase.

## Example

```ts
import { lowerCase } from 'funtool/string'

lowerCase('HeLLo')
// => 'hello'
```

## Parameters

- `input` (`string`): The string to convert.

## Returns

- `string`: The lowercase string.
