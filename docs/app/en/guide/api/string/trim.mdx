# `trim`

Removes whitespace from both ends of the `str`.

## Usage Scenarios

- **Data Cleaning**: In data processing, clean strings by removing leading and trailing whitespace.
- **Input Validation**: Validate user input by removing unnecessary whitespace.

## Example

```ts
import { trim } from 'funtool/string'

trim('  hello  ') // => 'hello' ✅
trim(123) // => '123' ✅
```

## Signature

```ts
function trim(str: string): string
```

## Parameters

- `str` ( `string` ): The input string.

## Returns

- ( `string` ): A new string with leading and trailing whitespace removed.
