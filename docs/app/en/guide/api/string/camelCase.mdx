# `camelCase`

Converts a string into `camelCase` format.

## Example

```ts
import { camelCase } from 'funtool/string'

camelCase('hello world')
// => 'helloWorld'
```

## Parameters

- `input` (`string`): The string to be transformed.

## Returns

- `string`: The camelCased string.
