# `union`

获取两个数组 `arr1` 和 `arr2` 的并集，返回一个新数组包含两个数组中的所有元素且不重复。

## 使用场景

- **数据聚合**：当需要合并两个数据集且不包含重复条目时。
- **列表合并**：合并两个列表同时确保元素的唯一性。
- **集合操作**：在数组上执行类似集合的并集操作。

## 示例

```ts
import { union } from 'funtool';

union([1, 2], [2, 3]); // ✅ [1, 2, 3]
union(['a'], ['b', 'a']); // ✅ ['a', 'b']
```

## 签名

```ts
function union<T>(arr1: T[], arr2: T[]): T[]
```

## 参数

- `arr1` (`T[]`)：第一个数组。
- `arr2` (`T[]`)：第二个数组。

## 返回值

- (`T[]`)：一个新数组，包含两个数组中的所有元素且不重复。
