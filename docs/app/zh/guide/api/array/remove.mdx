# `remove`

从数组 `arr` 中移除指定的 `values`，并返回一个新数组，其中不包含指定的值。

## 使用场景

- **数据清理**：当需要从数组中清理不必要的数据时。
- **结果过滤**：根据特定条件从数组中过滤出特定元素。
- **数组操作**：通过移除特定元素来修改数组。

## 示例

```ts
import { remove } from 'funtool';

remove([1, 2, 3], [2]); // ✅ [1, 3]
remove(['a', 'b', 'c'], ['b', 'c']); // ✅ ['a']
```

## 签名

```ts
function remove<T>(arr: T[], values: T[]): T[]
```

## 参数

- `arr` (`T[]`)：要处理的数组。
- `values` (`T[]`)：要移除的值。

## 返回值

- (`T[]`)：一个新数组，其中不包含指定的值。
