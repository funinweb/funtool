# `isNull`

检查给定的 `value` 是否为 `null`。

## 使用场景

- 空值检查：验证值是否特别为 null（不是 undefined 或其他假值）。
- 数据验证：验证 API 响应或用户输入。
- 类型安全：当 null 是有效情况时确保类型安全。

## 示例

```ts
import { isNull } from 'funtool';

isNull(null); // ✅ true
isNull(undefined); // ❌ false
isNull(0); // ❌ false
isNull(''); // ❌ false
```

## 函数签名

```ts
function isNull(v: any): boolean
```

## 参数

- `v` (`any`): 要测试是否为 null 的值。

## 返回值

- (`boolean`): 如果值为 null，则返回 `true`，否则返回 `false`。
