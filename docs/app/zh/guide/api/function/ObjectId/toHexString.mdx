# `toHexString()`

返回ObjectId的24字符十六进制字符串表示。

## 用法

```ts
import { ObjectId } from 'funtool';

const id = new ObjectId();
const hexString = id.toHexString(); // 例如 "5f1d7f3b1c9d440000000000"
```

## 签名

```ts
toHexString(): string
```

## 返回值

返回24字符的小写十六进制字符串。

## 注意事项

- 这是ObjectId的原生字符串表示
- 等同于调用`toString()`
- 始终返回小写十六进制字符
- 格式匹配MongoDB ObjectId标准
