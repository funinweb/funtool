# `toCamelCase`

将 `str` 转换为驼峰命名法（camelCase），即除第一个单词外，每个单词的首字母大写。

## 使用场景

- **变量命名**：在编程中，将字符串转换为驼峰命名法用于变量名。
- **数据标准化**：在数据处理中标准化字符串格式。

## 示例

```ts
import { toCamelCase } from 'funtool/string'

toCamelCase('hello_world'); // ✅ 'helloWorld'
toCamelCase('hello-world'); // ✅ 'helloWorld'
toCamelCase('api_version'); // ✅ 'apiVersion'
toCamelCase('API_version'); // ✅ 'apiVersion'
toCamelCase('my-xml-parser'); // ✅ 'myXmlParser'
```

## 函数签名

```ts
function toCamelCase(str: string): string
```

## 参数

- `str` ( `string` ): 输入的字符串。

## 返回值

- ( `string` ): 一个新的驼峰命名法格式的字符串。
