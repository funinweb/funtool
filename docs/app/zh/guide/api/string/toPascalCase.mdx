# `toPascalCase`

将 `str` 转换为帕斯卡命名法（PascalCase），即每个单词的首字母都大写。

## 使用场景

- **类命名**：在编程中，将字符串转换为帕斯卡命名法用于类名。
- **数据标准化**：在数据处理中标准化字符串格式。

## 示例

```ts
import { toPascalCase } from 'funtool/string'

toPascalCase('hello_world'); // ✅ 'HelloWorld'
toPascalCase('api_version'); // ✅ 'ApiVersion'
toPascalCase('API_version'); // ✅ 'APIVersion'
```

## 函数签名

```ts
function toPascalCase(str: string): string
```

## 参数

- `str` ( `string` ): 输入的字符串。

## 返回值

- ( `string` ): 一个新的帕斯卡命名法格式的字符串。
