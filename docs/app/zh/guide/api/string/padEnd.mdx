# `padEnd`

用 `padChar` 从 `str` 的末尾开始填充字符串，直到达到指定的 `length`。

## 使用场景

- **文本对齐**：在文本格式化中，将字符串对齐到特定宽度。
- **数据格式化**：在数据输出中，确保字符串具有一致的长度。

## 示例

```ts
import { padEnd } from 'funtool/string'

padEnd('hello', 10, '*'); // 'hello*****'
```

## 函数签名

```ts
function padEnd(str: string, length: number, padChar: string = ' '): string
```

## 参数

- `str` ( `string` )：输入字符串。
- `length` ( `number` )：字符串的目标长度。
- `padChar` ( `string` )：用于填充的字符。默认为空格。

## 返回值

- ( `string` )：填充后的字符串。
