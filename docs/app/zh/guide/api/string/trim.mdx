# `trim`

移除 `str` 两端的空白字符。

## 使用场景

- **数据清理**：在数据处理中，通过移除字符串首尾的空白字符来清理数据。
- **输入验证**：通过移除不必要的空白字符来验证用户输入。

## 示例

```ts
import { trim } from 'funtool/string'

trim('  hello  ') // => 'hello' ✅
trim(123) // => '123' ✅
```

## 函数签名

```ts
function trim(str: string): string
```

## 参数

- `str` ( `string` ): 输入字符串。

## 返回值

- ( `string` ): 一个新的移除了首尾空白字符的字符串。
