# `reverse`

反转 `str` 中字符的顺序。

## 使用场景

- **文本转换**：在文本处理中，出于各种目的反转字符串的顺序。
- **数据加密**：作为一种简单的数据加密或混淆形式。

## 示例

```ts
import { reverse } from 'funtool'

reverse('hello'); // 'olleh'
```

## 函数签名

```ts
function reverse(str: string): string
```

## 参数

- `str` ( `string` ): 输入的字符串。

## 返回值

- ( `string` ): 一个新的字符串，其中字符顺序已反转。
