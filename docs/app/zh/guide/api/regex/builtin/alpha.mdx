# `alpha`

用于验证字母字符的内置正则表达式插件。

## 使用场景

- **表单验证**：验证文本输入是否仅包含字母
- **数据清理**：确保字符串不包含数字或特殊字符
- **用户名验证**：验证用户名是否仅包含字母字符

## 示例

```ts
import { regex } from 'funtool';

// 基本验证
const isValid = regex.checker('abc').use('alpha').isValid();

// 带错误处理
const result = regex.checker('123').use('alpha');
if (!result.isValid()) {
  console.log('输入必须仅包含字母');
}
```

## 正则规则

```ts
/^[a-zA-Z]+$/ 
```

## 验证规则

- 必须仅包含字母 (a-z, A-Z)
- 不允许包含数字、空格或特殊字符
- 不区分大小写（同时接受大写和小写）

## 错误消息

对于无效输入返回 `false`，无特定错误消息。
