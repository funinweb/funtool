# `postal`

用于验证邮政编码的内置正则表达式插件。

## 使用场景

- **表单验证**：验证邮政编码输入字段。
- **地址验证**：在输入地址时确保邮政编码有效。
- **物流配送**：在处理发货前验证邮政编码。

## 示例

```ts
import { regex } from 'funtool';

// 基本验证
const isValid = regex.checker('123456').use('postal').isValid();

// 带错误处理
const result = regex.checker('invalid').use('postal');
if (!result.isValid()) {
  console.log('无效邮政编码');
}
```

## 正则规则

```ts
/\d{6}/
```

## 验证规则

- 输入必须是 6 位数字。

## 错误消息

对于无效的邮政编码，返回 `false`，不提供具体消息。
