# `qq`

用于验证 QQ 号码的内置正则表达式插件。

## 使用场景

- **表单验证**：验证 QQ 号码输入字段。
- **用户注册**：在注册时验证 QQ 号码。
- **联系人验证**：在添加联系人前确保 QQ 号码有效。

## 示例

```ts
import { regex } from 'funtool';

// 基本验证
const isValid = regex.checker('123456789').use('qq').isValid();

// 带错误处理
const result = regex.checker('01234').use('qq');
if (!result.isValid()) {
  console.log('无效 QQ 号码');
}
```

## 正则规则

```ts
/^[1-9][0-9]{4,9}$/
```

## 验证规则

- 必须以非零数字开头。
- 后面跟随 4 到 9 位数字。

## 错误消息

对于无效的 QQ 号码，返回 `false`，不提供具体消息。
